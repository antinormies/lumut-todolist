## Features

- **Jetpack Compose** – 100% declarative UI  
- **MVVM Architecture** – Clean separation of concerns  
- **Fullscreen Circular Loader** – Elegant loading state  
- **Read-only Detail View** – Shows `ID`, `Title`, `Completed`  
- **Strikethrough on completed** – Visual feedback  
- **No swipe-to-dismiss** – Pure display as requested
